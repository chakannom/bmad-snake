# BMAD Brief

## B: Business / Why
- 목표: 시간 제한 기반 스테이지 모드를 도입해 플레이 동기와 난이도 곡선을 강화한다.
- 성공 기준: 각 단계에서 제한 시간 내 목표 점수를 달성하면 다음 단계로 진행되고, 총 20단계를 연속 플레이 가능하다.

## M: Metrics
- `Stage Count`: 20개 스테이지 맵 제공
- `Progress Rule`: 제한 시간 내 목표 점수 달성 시 다음 단계 진입
- `Difficulty Curve`: 스테이지 상승에 따라 장애물/속도/요구 점수 난이도 증가
- `Bug count`: 치명 버그 0건
- `FPS`: 일반 노트북 기준 부드러운 체감(틱 기반)

## A: Assumptions
- 대상 플랫폼: 데스크톱 브라우저 우선
- 입력 장치: 키보드 방향키
- 초기 렌더링: Canvas 2D
- 스테이지 실패 조건은 시간 초과 또는 충돌
- 스페이스바는 현재 스테이지 재시작에 사용

## D: Delivery Scope (Stage MVP)
- 뱀 이동 / 성장
- 벽/자기 몸/장애물 충돌 판정
- 스테이지 점수/목표 점수/제한 시간 HUD
- 20개 맵 스테이지 구성
- 스테이지 클리어 시 다음 단계 자동 진입
- 사망 시 현재 스테이지 유지 + 점수 30% 차감 + 속도 초기화
