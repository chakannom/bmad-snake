# BMAD Brief

## B: Business / Why
- 목표: 2D 웹 스네이크 게임 MVP를 빠르게 출시한다.
- 성공 기준: 브라우저에서 즉시 실행 가능하고, 기본 게임 루프(이동/먹이/점수/충돌)가 동작한다.

## M: Metrics
- `Time to first playable`: 1일 이내
- `Bug count`: 치명 버그 0건
- `FPS`: 일반 노트북 기준 부드러운 체감(틱 기반)

## A: Assumptions
- 대상 플랫폼: 데스크톱 브라우저 우선
- 입력 장치: 키보드 방향키
- 초기 렌더링: Canvas 2D

## D: Delivery Scope (MVP)
- 뱀 이동 / 성장
- 벽/자기 몸 충돌 판정
- 점수 표시
- 재시작
